Configuration in Podium is very simple and straight forward.

Lets start with code structure:

  • Entire code for Podium is contained in src folder.
  • Open docs.config.js file you will find 2 structures.
    • docs: docs is an array of all the docs, create an id and docLabel, that construct your left navigation.
    • templates: templates is an json object it uses the id you create and map it with the html file name.
  • Now once you create a new id, docLabel and map your id to a template. Add the template file in partials/docs folder.
  • That's it.
