Lets get started with setting up your documentation, it will take less than 5 minutes to setup and then you can focus on your documentation.

Steps to follow:

  • Use command line interface(CLI) and go to your workspace
  • git clone git@github.com:colorgap/podium.git
  • cd podium
  • sudo npm install && bower install && gulp
  • gulp serve
We have added amazing browser-sync which helps you make changes to your docs and view right away. "gulp serve" will start a server which can be accessed at http://localhost:3000.
